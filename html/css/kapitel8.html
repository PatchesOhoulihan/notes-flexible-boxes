Browser Präfixe

Häufig ist die Spezifikation, einzelner CSS features noch nicht fertiggestellt, während es bereits Implementierungen für verschiedene Browser gibt. Diese unterscheiden sich zumeist erheblich. BSP: Die meisten Browser können Farbverläufe darstellen, jedoch ist die Syntax dafür sehr unterschiedlich. Daher sind die Browserherrsteller dazu übergegangen experimentelle Funktionen mit einem Browserpräfix zu maskieren.

Browser Browser-Präfix Layout Engine
Apple Safari -webkit- Webkit
Google Chrome -Webkit- Blink
Mozilla Firefox -moz- Gecko
Opera -o- Blink
Internet Explorer -ms- Trident

Beitrag zum Thema präfixes

normalize.css

Alle Browser besitzen ein eingebautes Stylesheet und benutzen verschiedene Voreinstellungen. Zur vereinheitlichung der verschiedenen Browser haben viele Entwickler diese durch einen Hardreset überschrieben. Dies ist jedoch zu umständlich dafür sollte man lieber die normalize.css verwenden
